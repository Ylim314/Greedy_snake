# 贡献指南

感谢你考虑为 Greedy Snake 项目做出贡献！

## 如何贡献

### 报告 Bug

- 使用 GitHub Issues 报告 bug
- 提供详细的复现步骤
- 包含你的操作系统和编译器信息

### 提交代码

1. Fork 项目
2. 创建功能分支
3. 编写代码并测试
4. 提交 Pull Request

### 代码规范

- 使用有意义的变量名
- 添加适当的注释
- 遵循现有的代码风格
- 确保代码能正常编译运行

## 开发环境设置

1. 安装 MSYS2 或 MinGW-w64
2. 配置 PDCurses 库
3. 克隆项目并编译测试

## 联系方式

如有问题，请通过 GitHub Issues 联系我们。
